Онлайн IQ-тест с оплатой за результат (MVP)

1. Общая информация

Тип проекта:
Онлайн-сайт с развлекательным IQ-тестом и оплатой за просмотр результата.

Цель проекта:
Быстро запустить MVP для рекламы и проверки спроса.
Проект не медицинский, не диагностический.

Формат реализации:
No-code / low-code, без кастомной разработки с нуля.

⸻

2. Общая логика работы сайта

Пользовательский путь:
1. Пользователь заходит на лендинг
2. Нажимает «Начать тест»
3. Проходит IQ-тест (20–25 вопросов)
4. Видит экран «Результат готов»
5. Оплачивает разовый платёж
6. Получает подробный результат
• на отдельной странице или
• по email

⚠️ Просмотр результата невозможен без оплаты.

⸻

3. Структура сайта

3.1 Лендинг (1 страница)

Обязательные блоки:
• Заголовок (описание теста)
• Краткое объяснение:
• 20–25 вопросов
• 10–15 минут
• логика + визуальные задания
• Кнопка «Начать тест»
• Дисклеймер:
«Тест носит развлекательный характер и не является медицинской или психологической диагностикой»
• Ссылки:
• Privacy Policy
• Terms of Use
• Cookie consent (базовый)

Платформа:
• Tilda / Webflow (или аналог no-code)

⸻

3.2 IQ-тест

Реализация через:
• Typeform / Tally / аналогичный сервис

Требования:
• 20–25 вопросов
• Типы вопросов:
• логические ряды
• числовые последовательности
• абстрактные фигуры
• часть вопросов с изображениями
• 4–6 вариантов ответа
• Вопросы отображаются по одному
• Индикатор прогресса
• Адаптация под мобильные устройства

В конце теста:
Сообщение:

«Ваш результат рассчитан. Для просмотра подробного отчёта необходимо оплатить доступ.»

⸻

4. Подсчёт результата
• Каждый вопрос имеет правильный ответ
• Подсчёт баллов автоматический
• 5–6 диапазонов результатов (уровни IQ)
• Разные тексты результатов для одинаковых диапазонов (по возможности)

❗ Числовой IQ и подробная интерпретация показываются только после оплаты

⸻

5. Оплата

Платёжная система:
• Stripe (обязательно)
• Разовый платёж
• Без подписки

После успешной оплаты:
• автоматический редирект или
• отправка email с результатом

⸻

6. Результат после оплаты

Формат (один из вариантов, по согласованию):
• закрытая страница с результатом
или
• email с текстом результата / PDF

Содержимое результата:
• диапазон IQ
• описание уровня
• краткая интерпретация
• сильные стороны мышления
• общие рекомендации

⸻

7. Минимальная защита от обхода

Обязательно:
• результат недоступен без подтверждённой оплаты
• прямой доступ по ссылке без оплаты невозможен
• защита реализуется стандартными средствами выбранных сервисов (без сложного кода)

⚠️ Идеальная защита не требуется, нужен рабочий MVP.

⸻

8. Аналитика

Подключить:
• Google Analytics
• Meta Pixel (если возможно)

Минимальные события:
• начало теста
• завершение теста
• успешная оплата

⸻

9. Что предоставляет заказчик

С моей стороны будет предоставлено:
• структура теста
• вопросы
• правильные ответы
• тексты результатов
• цена продукта
• домен

⸻

10. Что не входит в проект

Чтобы избежать недопонимания, в проект не входит:
• кастомная разработка backend
• личные кабинеты
• подписки
• сложная админ-панель
• мобильное приложение
• масштабируемая платформа

⸻

11. Сроки

Ожидаемые сроки:
• 3-4 недели

⸻

12. Результат сдачи проекта

Проект считается выполненным, если:
• тест корректно работает
• оплата проходит
• результат недоступен без оплаты
• все страницы открываются на мобильных
• переданы все доступы (Tilda / Stripe / формы)

⸻
